Lokesh Cinematic Universe

LCU is a movie universe created by Lokesh Kanagaraj. LCU is a fan given name.Director Loki has started this universe with his second film Kaithi where we can see South Actor Karthi as a main hero named Dilli.After that Lokesh came with Kamal Hassan's Vikram which was a blockbuster in box office.There we can see Kamal Hassn as main character named Vikram who wants a drug free society.Also in this movie we can see Suriya as Rolex who is the main villain of the universe.Last year,Lokesh came with his recently released film Leo where we can se Thalapathy Vijay as main character as Parthiban(Leo Das).In future Lokesh will come with few more movies like Kaithi 2,Rolex,Leo 2 and Vikram 2.Audience are expecting that Vikram 2 will be the end game of this movie universe.
